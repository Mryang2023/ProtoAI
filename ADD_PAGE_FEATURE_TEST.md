# 动态添加页面功能 - 测试指南

## 功能说明

在原型生成完成后，用户可以点击"添加新页面"按钮，输入页面名称和描述后，AI 会自动生成新页面并追加到现有页面列表末尾，同时保持与现有页面的样式一致性。

## 测试步骤

### 1. 启动开发服务器

如果服务器未运行：
```bash
npm run dev
```

访问 http://localhost:5180

### 2. 创建基础项目

1. 在左侧面板输入内容描述，例如："一个简单的博客网站"
2. 选择风格偏好（例如：现代 SaaS）
3. 选择目标平台（PC端/移动端/双端）
4. 点击"规划方案"
5. AI 分析后会显示页面规划（例如：首页、文章列表、文章详情）
6. 点击"确认方案，开始生成"
7. 等待所有页面生成完成

### 3. 测试添加新页面

当所有页面生成完成后（左侧面板显示"✓ X 页已生成"）：

1. **查看"添加新页面"按钮**
   - 应该出现在"保存为模板"按钮下方
   - 按钮样式：蓝色主色调，带 Plus 图标

2. **点击"添加新页面"按钮**
   - 展开一个表单区域
   - 包含：
     - 标题栏："新增页面" + 关闭按钮
     - "页面名称"输入框
     - "页面描述"文本域
     - "确认添加"和"取消"按钮

3. **填写表单**
   - 页面名称：输入"用户中心"
   - 页面描述：输入"展示用户个人信息、设置和收藏的文章"
   - 点击"确认添加"

4. **观察生成过程**
   - 按钮变为 loading 状态，显示"生成中..."
   - 右侧 iframe 实时显示 HTML 生成进度
   - 生成时间约 5-15 秒（取决于 AI 响应速度）

5. **验证生成结果**
   - ✅ 新页面自动出现在左侧页面列表末尾
   - ✅ 新页面标记为"✓"（已完成）
   - ✅ 右侧自动切换到新页面预览
   - ✅ 新页面的导航栏与现有页面一致
   - ✅ 配色方案与现有页面相同
   - ✅ 导航链接包含所有页面（包括新添加的）

### 4. 边界情况测试

#### 4.1 空输入验证
- 尝试不填名称或描述
- 预期："确认添加"按钮保持禁用状态

#### 4.2 连续添加多个页面
- 添加第一个页面后，再次点击"添加新页面"
- 添加第二个页面（例如："登录页面"）
- 预期：两个新页面都正常显示在列表中

#### 4.3 长名称处理
- 输入很长的页面名称（超过 50 字符）
- 预期：输入框自动截断

#### 4.4 特殊字符
- 页面名称包含空格、中文、数字
- 预期：路由自动生成，例如 `/用户-center-123`

### 5. 样式一致性验证

对比新生成的页面与原有页面：

1. **导航栏检查**
   - 位置：固定在顶部
   - Logo："ProtoAI"
   - 链接：包含所有页面
   - 当前页面高亮样式

2. **配色检查**
   - 背景色相同
   - 文字颜色相同
   - 主色调（按钮、链接）相同

3. **CSS 变量检查**
   - 打开浏览器开发者工具
   - 查看 `:root` 中的 CSS 变量
   - 应与首页完全一致

### 6. 交互测试

#### 6.1 页面切换
- 使用左右箭头键切换页面
- 点击左侧列表中的页面名称
- 点击导航栏中的链接
- 预期：都能正常切换到新页面

#### 6.2 代码编辑
- 切换到新页面
- 在代码编辑器中修改 HTML
- 预期：可以正常编辑和预览

#### 6.3 重新生成
- 点击新页面列表项中的"重新生成"按钮
- 预期：可以重新生成该页面

### 7. 导出测试

1. **导出为 ZIP**
   - 点击右上角"导出" → "导出全部为 ZIP"
   - 解压后检查是否包含新页面
   - 在新页面中点击导航链接，确认跳转正常

2. **导出为 React/Tailwind**
   - 尝试导出新页面为 React 组件
   - 尝试导出为 Tailwind 版本
   - 预期：都能正常转换

## 已知限制

1. **不支持删除页面**：添加后无法通过 UI 删除（未来扩展）
2. **不支持 reorder**：无法调整页面顺序（未来扩展）
3. **不更新 plannedPages**：新页面仅添加到 `pages` 数组，不改变原始规划
4. **需要已有首页**：如果没有生成任何页面，无法添加新页面（依赖参考模板）

## 常见问题

### Q: 为什么看不到"添加新页面"按钮？
A: 确保所有规划的页面都已生成完成。如果还在生成中或未完成，按钮不会显示。

### Q: 新页面的导航栏为什么和旧页面不一样？
A: 可能是 AI 没有严格遵循参考模板约束。可以尝试重新生成该页面。

### Q: 添加页面后可以重新规划吗？
A: 可以。重新规划会保留现有生成的页面，但会创建新的规划方案。
